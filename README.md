# Totem Hospitalar - Viver Bem

Este projeto foi desenvolvido como atividade de programação utilizando Python e GitHub.

O objetivo do sistema é simular um totem de autoatendimento de hospital, onde o paciente pode fazer um cadastro simples e escolher o tipo de atendimento desejado.

## Funcionalidades

* Cadastro do paciente (nome, idade e se possui deficiência)
* Classificação automática de prioridade
* Menu para escolher o serviço (consulta, exame ou emergência)
* Geração de senha de atendimento
* Aviso especial em caso de emergência
* Impressão de ticket no console
